# Crypto Listing App Built with React and Tailwind CSS

## 📌 Overview

This project is a **Crypto Listing App** built using **React** and **Tailwind CSS**. It allows users to view cryptocurrency details, track trending coins, and save their favorite cryptos for easy access.

## 🚀 Features

- 📊 **Crypto Screener** – Browse and filter cryptocurrencies in real-time.
- 🔍 **Detailed Coin View** – Get detailed information about each coin.
- 📈 **Trending Coins** – View the latest trending cryptocurrencies.
- 💾 **Saved List** – Save and manage favorite coins.
- 📱 **Mobile Responsive** – Fully optimized for all screen sizes using Tailwind CSS.

## 🛠️ Tech Stack

- **Frontend:** React.js, Tailwind CSS
- **Data Source:** CoinGecko API (or any other crypto API of choice)

## 📥 Installation & Setup

1. **Clone the repository:**
   ```sh
   git clone https://github.com/Bode683/Crypto-Listing-App.git
   cd Crypto-Listing-App
   ```
2. **Install dependencies:**
   ```sh
   npm install
   ```
3. **Start the development server:**
   ```sh
   npm start
   ```
4. **Open the app in your browser:**
   ```sh
   http://localhost:3000
   ```

## 📌 Usage

- Search for cryptocurrencies using the search bar.
- Click on a coin to view detailed information.
- View trending cryptos on the "Trending" page.
- Save your favorite cryptos for easy access.
